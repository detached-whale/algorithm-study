testdome javascript interview question no.10 (https://www.testdome.com/d/javascript-interview-questions/2)

Function appendChildren should add a new child div to each existing div. New divs should be decorated by calling decorateDiv.

For example, after appendChildren is executed, the following divs: <div id="a"> <div id="b"> </div> </div> should take the following form (assuming decorateDiv does nothing): <div id="a"> <div id="b"> <div></div> </div> <div></div> </div>
